# 神经网络

- 机器学习简介
    - 机器学习是什么---无序数据转化为价值的方法
    - 机器学习价值 --- 从数据中抽取规律,并用来预测未来
    - 机器学习应用举例
        - 分类问题 --- 图像识别、垃圾邮件识别
        - 回归问题 --- 股价预测、房价预测
        - 排序问题 --- 点击率预估、推荐
        - 生成问题 --- 图像生成、图像风格转换、图像文字描述生成
    - 机器学习岗位职责
        - 数据处理（采集+去噪）
        - 模型训练（特征+模型）
        - 模型的评估和优化(MSE、F1-score、AUC+调参)
        - 模型应用(A/B测试)
- 深度学习简介
    - 深度学习与机器学习
        - 机器学习是实现人工智能的方法
        - 深度学习是实现机器学习算法的技术
